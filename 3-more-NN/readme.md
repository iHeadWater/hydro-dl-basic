# 更多神经网络结构

除了最基本的ANN、RNN、CNN等，近年来，有很多更复杂先进的神经网络结构受到应用者关注。本文件夹下内容主要记录下这些常见的更复杂网络的基本概念。

目前主要包括RNN与ODE之间的微妙联系；以及一些state-of-the-art的RNN模型，如Transformer等；还有更多CNN相关的模型。
